_To Run Locally:_
Clone the repository and perform the following command line actions:
```bash
$ cd wordle
$ npm install
$ npm run start
```

_To build/run docker container:_
```bash
$ docker build -t focale .
$ docker run -d -p 3000:3000 focale
```
open http://localhost:3000 in browser.
